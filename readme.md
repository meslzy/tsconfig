# Shared typescript configuration  (✨)

> my personal shared typescript configuration

---

## Getting Started (✅)

- ### Installation (⏬)
  - `npm i @meslzy/tsconfig --save-dev`


- ### Usage  (🌠)

Add the following to your `tsconfig.json` file:

```json
{
  "extends": "@meslzy/tsconfig"
}
```
