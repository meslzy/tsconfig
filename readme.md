## Getting Started (✅)

- ### Installation (⏬)
  - `npm i @meslzy/tsconfig --save-dev`


- ### Usage  (🌠)

  - Base
    - `{"extends": "@meslzy/tsconfig/tsconfig.json"}`
  - React
    - `{"extends": "@meslzy/tsconfig/tsconfig.react.json"}`
